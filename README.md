🚀 AI-Powered SEO Rank Tracker
An AI-powered Full Stack SEO Rank Tracking platform that helps users monitor keyword rankings, analyze SEO performance, and automate SEO insights using modern web technologies and AI integration.

✨ Features
🔐 Secure User Authentication & Authorization
📈 Real-time Keyword Rank Tracking
🤖 AI-Powered SEO Insights using Gemini API
🌐 Automated Web Scraping with Playwright
⏰ Scheduled Rank Monitoring using Cron Jobs
📊 SEO Analytics Dashboard
📱 Fully Responsive UI
⚡ Fast and Optimized Frontend with Vite
🔄 RESTful API Architecture

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/your-username/your-repo-name.git
2️⃣ Navigate to Project Directory
cd your-repo-name
3️⃣ Install Dependencies
Frontend
cd client
npm install
Backend
cd server
npm install
🔑 Environment Variables
Create a .env file inside the server directory and add:

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
GEMINI_API_KEY=your_gemini_api_key
▶️ Running the Application
Start Backend Server
cd server
npm run dev
Start Frontend
cd client
npm run dev
